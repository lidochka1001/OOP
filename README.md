# OOP
1. Инофрмационная система "Поликлиника"
2. Классы: Отделение, Врачи, Пациенты, Приемы, Карточки пациентов

Отделение:
  - ID отделения
  - Название отделения
  - Этаж отделения

Врачи:
  - ID врача
  - Фамилия
  - Имя
  - Отчество
  - Специальность
  - Кабинет приема

Пациенты:
  - ID пациента (номер полиса)
  - Фамилия пациента
  - Имя пациента
  - Отчество пациента
  - Адрес проживания
  - Телефон
  - Дата рождения

Прием:
  - ID приема
  - ID отделения
  - ID врача
  - ID пациента
  - Дата приема
  - Примечание

Мед карта:
  - ID мед карты
  - ID пациента
  - Диагноз
  - История болезни
